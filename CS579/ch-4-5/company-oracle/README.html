There are three steps to creating the company schema:
To run these scripts, you must log in to sqlplus as a user with the DBA role.
Create the user
run @create_user. This creates a user company with password CS579, and grants the appropriate roles.
Create the tables
First, make sure to log on as the new company user: 
conn company/CS579@connect_string
.
Next, run @createall.
Load the data
Finally, run @loadall
